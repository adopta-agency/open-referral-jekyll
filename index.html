Jekyll Open Referral

This is my working prototype for a human services website and API that runs on Github. My goal is to create a data store, website, and API that can be forked and operated by anyone looking to deploy a website.

These organizations and locations are pulled from the JSON files stored in the _data folder for this Github Pages driven, Jekyll website.

The goal is to create a website that is free to launch, meeting the human needs of the human services data, while also providing in a machine readable way that anyone can use in other web, mobile, and system to system integrations.

I am just getting going with this project and will be publishing the story, and all the data here in the Github repository -- you can also use the Github issues for any comments or feedback.
